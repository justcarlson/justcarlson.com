<task type="auto">
  <name>Task 1: Regenerate favicon.ico from SVG</name>
  <files>public/favicon.ico</files>
  <action>
Use ImageMagick to convert favicon.svg to favicon.ico with multiple resolutions:

```bash
cd /home/jc/developer/justcarlson.com
magick -density 384 public/favicon.svg \
  -background none \
  -define icon:auto-resize=48,32,16 \
  public/favicon.ico
```

Parameters:
- `-density 384` - High DPI rendering from SVG (384 = 96 DPI * 4x for sharpness)
- `-background none` - Preserve transparency
- `-define icon:auto-resize=48,32,16` - Bundle three standard sizes in one ICO

Expected output: favicon.ico under 10KB with JC monogram at all sizes.

Note: The SVG has CSS media queries for dark/light mode, but ICO format doesn't support this.
The ICO will render with the default (light mode) colors. This is expected behavior for
legacy favicon.ico - modern browsers use favicon.svg which does support dark mode.
  </action>
  <verify>
1. `ls -la public/favicon.ico` shows file size under 10KB
2. `file public/favicon.ico` shows "MS Windows icon resource" with multiple sizes
  </verify>
  <done>favicon.ico regenerated from JC monogram SVG, file size reduced from 103KB to under 10KB</done>
</task>
